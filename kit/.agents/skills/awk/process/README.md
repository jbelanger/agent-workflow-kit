# AWK Process Skills

Use this category for AWK workflow operations:

- initialization and maintenance (`init-awk`, `maintain-awk`),
- backlog and continuation routing,
- grooming and early vision discovery,
- durable artifact drafting and review,
- breakdown,
- implementation preparation,
- local work routing,
- review routing,
- workflow improvement.

Do not put code-quality, domain-modeling, or tool-specific specialist skills here unless they are
part of the process loop.
